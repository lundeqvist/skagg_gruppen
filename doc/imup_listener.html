Function Details

listen/3

listen(ReceiverPID, Port, HandlerPID) -> Pid | {error, Error}

To be called by the main server process to be able to listen to port Port and HandlerPID will receive the messages. If listen/2 fails with Reason, the function returns {error,Reason}. If successful it will return the Pid of the socket.

Example

  HandlerPID = Process which can handle messages
  1> ServerSocket = imup_listen(5555, HandlerPID)
  #Port<0.669>
